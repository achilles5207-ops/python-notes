Scope & Objective -:


The objective of this course is to develop programming skills in Python which is rich in tools & libraries and is popularly used for
solving real-life computing problems in many engineering domains.


Pre-Requisite -:


Basics of programming, algorithms and problem solving skills are required. Prior experience with a programming language will
be beneficial.
